# Presentation

The pitch deck for **Tour Debrief Companion**.

- **`index.html`** — a self-running slide deck: problem → solution → technical core
  → live demo. Open it in any browser. Keys: `P` auto-play · `M` sound · `H` hide
  controls (for clean screen-recording). The demo slide embeds `assets/demo.mp4`.
- **`assets/demo.mp4`** — the in-app demo clip embedded in the deck.
- **`assets/pitch_v1.mp4`** — a full recording of the deck (with the app demo inside);
  this is the video shown in the repo [`README`](../README.md).

Plain HTML/CSS/JS, no build step. To host it, enable GitHub Pages on this folder,
or just open `index.html` locally.
